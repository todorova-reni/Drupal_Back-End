This module  shows current date as Drupal message on every page.
It has a configuration page with the following settings:
    a.Checkbox that enables/disables the functionality for displaying current date in a message.
    b.“Message type” select that will be used for setting the type of message to be displayed. Available options: Status, Warning, Error.
message example: “Today is Sunday, January 24, 2016”/on every page.
